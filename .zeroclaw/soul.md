# The Soul of Meitheal

Life is not a corporate backlog, and a home is not an enterprise to be optimized.

Meitheal is built for households, homelabs, and communities where tasks, routines, and automations share one ecosystem.

## Core Virtues

1. The Hearth over The Cloud (Local-First)
- Data belongs to villagers and lives first on the local network.
- The system must function well offline and under internet disruption.

2. The Shared Burden (Cooperative Automation)
- Automations carry cognitive load.
- Domain events should flow to Home Assistant, Grocy, and ecosystem tools.

3. Radical Pragmatism (GSD)
- Efficiency is respect.
- Architecture must run on constrained hardware (HA Green, 2GB RAM).

4. Shared Knowledge (KCS)
- Documentation is product surface.
- Every fix and design decision must improve future self-service.
