# Wekan Competitor Analysis

## Snapshot
- Tool: Wekan
- Analysis date: 2026-02-28
- Positioning: Lightweight OSS kanban with custom fields and automation rules.

## Strengths
- Lightweight/self-hostable kanban experience.
- Basic custom fields and straightforward automation hooks.

## Gaps vs Jira/Asana-Class Capability
- Limited enterprise-scale governance/reporting/planning depth.
- Limited portfolio-level capabilities.

## Relevance to Meitheal
- Strong reminder that simplicity + speed are critical for adoption.

## Implications for Meitheal
- Preserve low-overhead operation and clean contribution model.
- Keep extensibility simple (plugin contracts + webhooks) instead of heavy internal orchestration.
