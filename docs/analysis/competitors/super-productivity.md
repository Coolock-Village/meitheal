# Super Productivity Competitor Analysis

## Snapshot
- Tool: Super Productivity
- Analysis date: 2026-02-28
- Positioning: Personal execution engine with strong local/offline focus, timeboxing, and issue integrations.

## Strengths
- Strong personal productivity ergonomics and local-first/offline behavior.
- Timeboxing/time-tracking workflows are highly polished.
- Mature developer-oriented integrations (for personal execution contexts).

## Gaps vs Jira/Asana-Class Capability
- Not a full multi-user project operating system with centralized governance model.
- Collaboration/admin controls are weaker than enterprise PM suites.
- Central portfolio/program management is limited compared with Jira/Asana-class tooling.

## Relevance to Meitheal
- Primary parity target for local-first speed, cognitive-load reduction, and personal workflow quality.

## Implications for Meitheal
- Preserve low-latency local-first execution as a hard constraint.
- Match/exceed personal execution UX while keeping household/team control plane via HA runtime.
- Add portfolio/workload/goals capability without sacrificing HA Green footprint.
