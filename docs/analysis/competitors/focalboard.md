# Focalboard Competitor Analysis

## Snapshot
- Tool: Focalboard
- Analysis date: 2026-02-28
- Positioning: Lightweight board-centric OSS PM for small-team/personal use.

## Strengths
- Simple board UX and low friction setup.
- Good baseline for straightforward task collaboration.

## Gaps vs Jira/Asana-Class Capability
- Limited advanced portfolio/governance depth.
- Standalone maintenance concerns affect long-term confidence.

## Relevance to Meitheal
- Reinforces need for clean, maintainable architecture and clear governance docs.

## Implications for Meitheal
- Keep contribution model and repo structure extremely clear.
- Preserve modular architecture so platform can evolve without lock-in.
