# Leantime Competitor Analysis

## Snapshot
- Tool: Leantime (OSS)
- Analysis date: 2026-02-28
- Positioning: Broad PM + strategy-oriented OSS platform with neurodiversity-aware positioning.

## Strengths
- Broad feature surface with strategy-oriented framing.
- Productivity/ADHD-oriented narrative aligns with Meitheal goals.

## Gaps vs Jira/Asana-Class Capability
- Some advanced strategy/program/AI features are not fully OSS-core.
- Platform breadth can come with complexity cost.

## Relevance to Meitheal
- Useful benchmark for ADHD-friendly workflow framing and communication.

## Implications for Meitheal
- Keep ADHD-aware UX and low-cognitive-load defaults explicit in requirements.
- Maintain KCS quality so workflows remain discoverable and self-service.
