# ADR 0003: Clean-Room Protocol

## Status
Accepted

## Date
2026-02-28

## Rules

- `meitheal-core` contributors must not copy upstream Vikunja code/assets.
- API behaviors may be informed by public docs/specs only.
- Any upstream code reuse must occur only inside `meitheal-vikunja-adapter` with AGPL notice.
- PR template must include a clean-room declaration checkbox for `meitheal-core` changes.
