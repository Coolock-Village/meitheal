# 90-Day Roadmap

## Phase 1 (Days 1-30): Core Vertical Slice + Governance

Acceptance criteria:
- DDD monorepo scaffold and governance CI in place.
- Task creation with framework fields persisted.
- Domain event emitted and audit logged.
- HA ingress middleware operational.

## Phase 2 (Days 31-60): HA/Grocy Automations

Acceptance criteria:
- Grocy and calendar connectors functional.
- Node-RED/n8n webhook flows documented and tested.
- Observability dashboards and core alerts provisioned.

## Phase 3 (Days 61-90): Plugin Platform + Analytics

Acceptance criteria:
- Plugin contract stabilized.
- RICE/DRICE/HEART analytics views available.
- Performance and reliability budgets gated in CI.
