# Plan 42-01: Fix 5 Ingress Bugs

## Objective

Resolve all 5 root causes of HTTP errors behind HA Supervisor ingress.

## Tasks

1. **Double prefix guard** — `Layout.astro` fetch interceptor checks `!input.startsWith(ip)` before prefixing
2. **CSRF bypass** — `middleware.ts` skips origin check when `ingressPath` is truthy
3. **Health check credentials** — `connectivity.ts` adds `credentials: "include"`
4. **CSP update** — `data:` in `script-src`, `ws: wss:` in `connect-src` for ingress
5. **Code cleanup** — Remove unused allowedHosts variable

## Verification

- Build passes (`pnpm --filter @meitheal/web build`)
- No 401/403/404 in browser console when accessing via ingress
- CRUD operations work through ingress
