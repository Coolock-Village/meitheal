# Wave 2: P1 UX Polish

## Goal

Transform the board from prototype feel to production-grade with hover states, drag feedback, search filtering, visual consistency.

## Tasks

### T-1906: Card Hover & Interaction States

Add CSS hover state to `.kanban-card`: subtle shadow elevation, border glow, smooth transition. Add `.dragging` class visual treatment (opacity 0.5, dashed border). Add `.drag-over` class on columns (highlight border or background lightening).

**Files:** `global.css`
**Verify:** Hover card → shadow lift. Drag card → ghosted appearance. Drag over column → column highlights.

### T-1907: Search Filters Kanban Cards

Wire the existing search input to filter cards on the board in real-time:
1. Listen to `input` event on search bar
2. Filter `.kanban-card` elements by `data-title` matching
3. Hide non-matching cards with `display: none`
4. Update column counts to reflect visible cards only

**Files:** `kanban.astro` (script section)
**Verify:** Type in search → only matching cards visible, counts update.

### T-1908: Consistent Add-Task Buttons

Unify all `+ Add Task` buttons to use the same `.kanban-add-btn` class. Custom lane add buttons should use identical styling to built-in columns. Remove the inconsistent bold styling on custom lane buttons.

**Files:** `kanban.astro` (lane rendering), `global.css`
**Verify:** All column add buttons look identical regardless of built-in vs custom lane.

### T-1909: Styled Board Selector

Replace stock `<select>` with custom dropdown matching dark theme. Use CSS-only approach: styled wrapper with chevron icon, matching background/border colors, rounded corners.

**Files:** `global.css`, `kanban.astro` (or Layout.astro sidebar)
**Verify:** Board selector matches dark theme, no stock browser styling visible.

### T-1910: Description & Card Layout Polish

Style `.kanban-card-desc` with muted color, smaller font, max 2 lines with text-overflow ellipsis. Add visual separator between title and description. Tighten card padding for density.

**Files:** `global.css`
**Verify:** Cards show descriptions in muted, truncated style with clear hierarchy.

## Verification Gate

- Card hover: visible shadow lift transition
- Drag feedback: opacity change + column highlight
- Search: real-time card filtering
- Visual consistency: all add buttons match
- Board selector: styled dark theme
- Description: muted, truncated, 2-line max
