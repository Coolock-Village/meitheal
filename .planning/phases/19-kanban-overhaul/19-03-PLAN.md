# Wave 3: P2 Competitive Parity

## Goal

Reach Trello/Linear competitive feature parity: due-date badges, label colors, WIP limits, keyboard navigation, empty state polish.

## Tasks

### T-1911: Due-Date Badges on All Cards

Show due-date badge on every card that has a `due_date` value. Use color coding: overdue (red), due today (amber), upcoming (green), no date (hidden). Already partially implemented — ensure consistent display.

**Files:** `kanban.astro` (card template), `global.css`
**Verify:** Cards with due dates show colored badge, overdue tasks highlighted red.

### T-1912: Label Color Badges

Parse label data and display as small colored chips on cards. Use label hex_color from the labels API. Position under the card title/description area.

**Files:** `kanban.astro` (card template), `global.css`
**Verify:** Tasks with labels show colored badge chips matching label colors.

### T-1913: WIP Limits

Add optional task limit per column. Display warning state (amber header) when at limit, blocked state (red) when over limit. Store limits in a column config. Show "3/5" format in column header count badge.

**Files:** `kanban.astro` (column header + script), `global.css`
**Verify:** Set WIP limit → header shows usage, amber at limit, red over.

### T-1914: Keyboard Navigation

Add arrow key navigation between cards within and across columns:
- `→` / `←` — move focus between columns  
- `↑` / `↓` — move focus between cards in column
- `Enter` — open task detail panel
- `Space` — toggle task status (move to next column)

**Files:** `kanban.astro` (script section)
**Verify:** Tab into board, use arrow keys to navigate, Enter opens detail.

### T-1915: Empty State & Column Collapse

Upgrade "Drop tasks here" to styled empty state with icon and encouragement text. Add column collapse toggle (click header to minimize column to icon-only width).

**Files:** `kanban.astro` (column template + script), `global.css`
**Verify:** Empty columns show styled state. Click column header → collapses to narrow bar.
