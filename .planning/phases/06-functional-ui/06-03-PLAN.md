# Phase 6 — Wave 3: Kanban + Table Views + Filters

## Scope
Alternative task views and advanced filtering — exceeds Vikunja with kanban drag-drop and spreadsheet table.

## Tasks

### T-621: Kanban Board Page
- **File:** `apps/web/src/pages/kanban.astro`
- 3 columns: Pending, Active, Complete
- Drag-and-drop between columns (native HTML5 drag API, no dependencies)
- Card shows: title, priority badge, due date, labels
- Click card → edit modal
- "+" button at top of each column for quick-add
- Column task count badges

### T-622: Table View Page
- **File:** `apps/web/src/pages/table.astro`
- Spreadsheet-style sortable table
- Columns: checkbox, title, status, priority, due_date, labels, actions
- Click cell to edit inline
- Column header click to sort (asc/desc with arrow indicator)
- Bulk select via checkboxes → bulk status change / delete

### T-623: Filter System
- Filter chips below search bar
- Filters: status, priority, due date range, labels, assignee
- Multiple filters can be combined (AND logic)
- Clear individual filters or "Clear all"
- URL state for shareable filtered views (query params)

### T-624: View Switcher
- Toggle buttons: List | Kanban | Table
- Persists selection in localStorage
- Smooth transition between views
- Same data, different presentation

## Verification
- Drag-and-drop updates task status
- Table inline editing works
- Filters combine correctly
