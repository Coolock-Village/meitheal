# Plan 05-02: Domain Events + Structured Logging + Shared Types

**Wave:** 2
**Scope:** Worker domain events, structured logger, shared types package, sync queue TTL

## Tasks

### T-511: Shared Types Package (Audit #50)
- Create `packages/shared-types/` with common types used by both runtimes
- Move `Task`, `SyncOperation`, `DomainEvent` interfaces into shared package
- Both Worker and Astro import from `@meitheal/shared-types`

### T-512: Worker Domain Events (Audit #34)
- Worker CRUD operations emit domain events
- Events: `task.created`, `task.updated`, `task.deleted`
- Inline event emitter (no Node-only deps) posts to integration bus endpoint

### T-513: Structured Logging (Audit #32)
- Port compat logger pattern to Workers-compatible implementation
- JSON structured logs with `level`, `message`, `timestamp`, `context`
- Replace all `console.error` with structured logger calls

### T-514: Sync Queue TTL Cleanup (Audit #12)
- Add `cleanupOldEntries()` to offline-store
- Remove pending sync operations older than 7 days
- Run on every sync cycle and on app startup

### T-515: Tests
- Shared types import from both runtimes
- Domain events emitted on CRUD
- Structured logger output format
- TTL cleanup removes expired entries

**Total:** 5 tasks
