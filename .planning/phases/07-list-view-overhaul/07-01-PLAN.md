# Phase 7 Plan 01: Table Row Grouping + Extended Filters

## Objective

Add Teamhood-style collapsible row grouping (swimlanes) to the table/list view and extend filter dimensions to match kanban (type, board, group-by). This is Phase 1 of the list view overhaul.

## Context

- @apps/web/src/lib/filter-state.ts — shared filter persistence
- @apps/web/src/pages/table.astro — table view page
- @apps/web/src/lib/table-controller.ts — table client-side logic

- @.planning/research/ARCHITECTURE.md — architecture decisions
- @.planning/research/PITFALLS.md — known pitfalls
