<task type="auto">
  <name>Task 2: Add board filter to Kanban toolbar</name>
  <files>apps/web/src/pages/kanban.astro</files>
  <action>
    Add a board filter dropdown to the Kanban toolbar (the filter bar area that contains
    the group-by select, density select, etc.). This addresses the P1 bug about board
    filtering not being well-integrated with the Kanban.

    1. In the toolbar/header area (~line 197-257), add a `<select>` element for board filtering:
       ```html
       <select id="boardFilterSelect" class="form-select" style="min-width: 140px;">
         <option value="">All Boards</option>
       </select>
       ```

    2. In the JS init section, populate the board filter on load:
       ```javascript
       const boardFilterSelect = document.getElementById("boardFilterSelect");
       if (boardFilterSelect) {
         fetch((window.__ingress_path || "") + "/api/boards")
           .then(r => r.json())
           .then(data => {
             (data.boards || []).forEach(b => {
               const opt = document.createElement("option");
               opt.value = b.id;
               opt.textContent = `${b.icon || '📋'} ${b.title}`;
               boardFilterSelect.appendChild(opt);
             });
           });
       }
       ```

    3. Add change handler that filters cards by `data-board` attribute:
       ```javascript
       boardFilterSelect?.addEventListener("change", () => {
         const boardId = boardFilterSelect.value;
         document.querySelectorAll(".kanban-card").forEach(card => {
           if (!boardId || card.dataset.board === boardId) {
             card.style.display = "";
           } else {
             card.style.display = "none";
           }
         });
         // Sync with sidebar
         document.dispatchEvent(new CustomEvent("board-filter-change", {
           detail: { boardId }
         }));
       });
       ```

    4. Listen for sidebar board changes to sync the toolbar:
       ```javascript
       document.addEventListener("board-changed", (e) => {
         if (boardFilterSelect && e.detail?.boardId) {
           boardFilterSelect.value = e.detail.boardId;
           boardFilterSelect.dispatchEvent(new Event("change"));
         }
       });
       ```
  </action>
  <verify>
    - `npm run build` passes
    - `grep -n "boardFilterSelect" apps/web/src/pages/kanban.astro` shows the new filter code
  </verify>
  <done>
    Board filter is integrated in the Kanban toolbar and syncs with sidebar.
    Users can filter cards by board directly from the Kanban view.
  </done>
</task>
