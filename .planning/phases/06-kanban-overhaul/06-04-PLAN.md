    In kanban.astro, find the swimlane toggle click handler (search for "swimlane-toggle" or
    "swimlane-header" click) and update it to:
    1. Toggle `.collapsed` class on `.swimlane-cards` sibling
    2. Toggle `.collapsed` class on the toggle arrow (`.swimlane-toggle`)
    3. Store collapse state in localStorage per swimlane type

    If no click handler exists yet, add one in the script section:
    ```javascript
    document.querySelectorAll(".swimlane-header").forEach(header => {
      header.style.cursor = "pointer";
      header.addEventListener("click", () => {
        const cards = header.nextElementSibling;
        const toggle = header.querySelector(".swimlane-toggle");
        const type = header.dataset.type;
        const key = `meitheal-kanban-swimlane-${type}`;
        const isCollapsed = cards.classList.toggle("collapsed");
        toggle?.classList.toggle("collapsed", isCollapsed);
        localStorage.setItem(key, isCollapsed ? "collapsed" : "expanded");
      });
      // Restore saved state
      const type = header.dataset.type;
      const saved = localStorage.getItem(`meitheal-kanban-swimlane-${type}`);
      if (saved === "collapsed") {
        header.nextElementSibling?.classList.add("collapsed");
        header.querySelector(".swimlane-toggle")?.classList.add("collapsed");
      }
    });
    ```
  </action>
