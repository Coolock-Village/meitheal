# Phase 55-03: Settings UI — Entity Registration Toggle

## Goal

Add a "Meitheal HA Entities" section to Settings → Integrations that lets
users enable/disable entity registration with a clear explanation of what
each entity does.

## Tasks

### 55-03-01: Create API route /api/ha/component

Endpoint to check component install status:

- GET: returns { installed: bool, version: string, entities: string[] }
- POST: { action: "install" | "uninstall" } — triggers component copy/remove

### 55-03-02: Add Settings UI section

Add "HA Entity Registration" section to SettingsIntegrations.astro:

- Status badge: Installed/Not Installed
- Entity list with descriptions (what each entity does)
- Enable/Disable toggle
- Auto-detected status message

### 55-03-03: Client-side JS wiring

In settings.astro, add hydration for the entity registration section:

- Fetch /api/ha/component on page load
- Update status badge and entity list
- Handle toggle save
