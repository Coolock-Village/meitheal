# Phase 22-01: AI Context Generation & Routing

## Status: Draft

## Objective

Provide a seamless integration between Meitheal tasks and configurable LLM providers (ChatGPT, Claude, Gemini, Local Ollama) to assist users in planning, debugging, and decomposing work. Satisfies the placeholder "Ask AI" settings integration.

## Scope

- **Context Service:** Create `ai-context-service.ts` to extract the selected task's title, description, custom fields, and status, and format them into a deterministic LLM prompt.
- **UI Integration:** Add an "Ask AI 🤖" button inside the Task Detail panel (`Layout.astro`), next to the Delete and Save buttons.
- **Clipboard Sync:** Automatically copy the generated context prompt to the user's clipboard using the `navigator.clipboard` API to bypass provider-specific URL length restrictions and ensure complex context isn't truncated.
- **Routing:** Open a new tab (`target="_blank"`) to the configured provider URL (e.g., `https://chatgpt.com/`, `https://claude.ai/new`, or a custom local endpoint).
- **Feedback:** Show a toast notification indicating "Context copied to clipboard. Opening [Provider]...".

## Verification

- Clicking "Ask AI" copies a formatted Markdown payload to the clipboard and opens the correct service.
- The custom endpoint URL works specifically for Local Ollama / custom hosting.
